## test with redis real queue

first run redis with persistence
`docker run -p 6379:6379 --name redis -d redis --save 60 1 --loglevel warning`

stop service
`docker stop redis`

start service
`docker start redis`

copy env values
`cp .env.example .env`

fill in your redis credentials if they are not `localhost:6379`
